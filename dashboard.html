Architecture Strip

Worker phone call → Bolna voice agent → Saathi router → Specialist tools → Cartesia voice card / Case dashboard / Follow-up call

Worker Phone Call

Simple phone access for migrant workers who need help without app installs or forms.

Bolna Voice Agent

Voice orchestration, telephony, call webhooks, custom tool invocations, and outbound follow-up.

Saathi Router

Warm Hindi front-door experience that gathers context and routes to the right assistance flow.

Specialist Tools

Wage complaint drafting, hospital voice support, and future paperwork or housing tools.

Cartesia Voice Card

Natural multilingual TTS for Kannada/Hindi speech cards, with timestamps when available.

Case Dashboard

Judge-friendly command center for state, transcript, case updates, and follow-up readiness.

Single-file dashboard served directly by FastAPI. No client-side framework required.
